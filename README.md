# snake 🐍

A small terminal Snake game written by hand in Rust 🦀

## Run ▶️

```bash
cargo run
```

## Controls 🎮

- Arrow keys: move ⬅️⬆️➡️⬇️
- `Space`: pause/resume ⏸️
- `Esc`: quit 🚪
